<h3>牛肉燴三絲</h3>

<img src="003.jpg" width="400" height="300" align="left">

用油10克起鍋，濺入紹酒，注入二湯、上湯。<br>
將牛肉絲、筍絲、冬菇絲放在鍋中，用精鹽、味素調味。<br>
待滾，用深色醬油調色， <br><br>
